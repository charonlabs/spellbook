Get a grounded answer with citations from web search.

Asks a question and returns a synthesized answer backed by web sources. Each claim is grounded in cited URLs. Best for quick factual lookups where you need a direct answer rather than raw search results.

Usage:
- `WebAnswer(query="What is the architecture of Gemma 4?")` — factual lookup
- `WebAnswer(query="When was Python 3.12 released and what were the major features?")` — specific question
- `WebAnswer(query="What are the differences between Flash Attention 1 and 2?")` — comparative question

Tips:
- Best for factual questions with definitive answers.
- The answer includes source citations — verify claims against the cited URLs if accuracy is critical.
- For exploratory research, use WebSearch instead — it gives you the raw sources to analyze yourself.
- For reading a specific document you've already found, use WebRead.
